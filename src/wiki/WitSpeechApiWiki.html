Configuring Meta Wit Speech to Text

Platform Overview - What is Meta Wit Speech to Text?

Wit.ai® Speech to Text technology enables fyou to accurately convert speech into text with an API powered by the best of Meta’s AI research and technology.

Getting an API key

  • Go to wit.ai and sign up for free with your Facebook account
  • Press Continue With Facebook and connect to Wit.ai with a Facebook account
  • Go to wit.ai/apps and click on New App
  • Choose a name and select a language (English works best), set the app visibility to private, then press CREATE
  • Go to Management > Settings (https://wit.ai/apps//settings) ENABLE API
  • Copy the API key from the Client Access Token section

Configuring the Nocapthca extension

  • Open the extension's options
  • Under Speech recognition select Wit.ai
  • Under API language select the language you have picked for the Wit.ai app, then press Add API
  • Paste the API key you have previously copied from Wit.ai into the API key: language field
