Configuring IBM Watson Speech to Text

Platform Overview - What is IBM Watson Speech to Text?

IBM WatsonÂ® Speech to Text technology enables fast and accurate speech transcription in multiple languages for a variety of use cases.

Getting an API key

  • Go to IBM Cloud , create an account and log in
  • Go to Speech to Text, select a location for the service, select the free pricing plan, and accept the terms of service, then click on Create
  • After the service is created, go to Manage and click on Show credentials
  • Copy the displayed API key and API endpoint URL

Configuring the Nocapthca extension

  • Open the extension's options
  • Under Speech recognition select IBM Watson Speech to Text
  • Paste the API endpoint and API key you have previously copied from IBM Cloud into their respective fields
