Configuring Microsoft Azure Speech to Text

Platform Overview - What is Microsoft Azure Speech to Text?

Microsoft Azure® Speech to Text technology enables fyou to accurately convert speech into text with an API powered by the best of Microsoft’s AI research and technology.

Getting an API key

  • Go to Microsoft Azure and sign up for free with a Microsoft account, you may need to add a credit card to activate the account
  • Go to Create Speech Servicesto create a new speech service, create a new resource group, such as cognitive-services, select a region, give the service a unique name, select the free pricing tier, and click on Review + create to validate the provided details, then click on CREATE and wait for the service to be created, then select the project after it has been created
    • Go to and click on Show Keys Copy the first displayed API key
  • Go to Cognitive Services and click on the newly created service
  • Go to Keys and Endpoint and click on Show Keys
  • Copy the first displayed API key

Configuring the Nocapthca extension

  • Open the extension's options
  • Under Speech recognition select Microsoft Azure Speech to Text
  • Paste the API endpoint and API key you have previously copied from Microsoft Azure into their respective fields
